# LINE CAREERS Clone Coding
[LINE CAREERS](https://careers.linecorp.com/ko/) 클론 코딩 프로젝트
HTML, CSS, JavaScript를 이용하여 LINE CAREERS 페이지 구현 프로젝트

## 목적
+ HTML 레이아웃 작성 방법을 익힌다.
+ 웹 표준과 웹 접근성을 준수한다. 

## 사용 기술
+ HTML
+ CSS
+ JavaScript

## 프로젝트 명세
+ 스크롤을 일정 범위 내렸을 때 헤더의 스타일이 변경된다.
+ search 버튼 클릭시 header-select가 활성화 되고, header-select 이외의 부분은 dimmed 처리로 비활성화 한다.
+ 셀렉트 박스는 한 페이지 내에서 한 번에 하나의 셀렉트 박스만 활성화 된다.
+ career section 내의 셀렉트 박스 옵션 선택시 버튼이 활성화 된다.
